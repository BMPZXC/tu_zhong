## 🎮 核心内容

🧩 《途中》讲述了一次发生在“桥接系统”中的异常任务。

你将作为**攻击程序**接入一个持续报错的网络，

与名为“电子幽灵”的存在协同行动，

修复一条始终无法完成的连接通路。

桥被不断标记，被反复重建，却从未真正抵达对端。 

身体被替代，情绪被隔离，路径在中途崩塌。 

🖤 **这是一个关于「替代」「下沉」与「无法抵达」的故事。**

**当所有路径都指向连接，你是否真的知道，自己正在走向哪里？**

- ✅ 剧情：**完全原创**
- ✅ 战斗：**原创横版战斗系统**
- ✅ 题材关键词：桥 / 连接失败 / 身体替代 / 情绪下沉 / 对端缺失

## 🕹 操作与战斗结构

游戏采用 **横版即时战斗 + 叙事推进** 的复合结构：

- A / D：角色移动
- W：跳跃（两段））
- 鼠标左键：攻击（剑,弓,盾）
- 鼠标右键：闪避  
 
每一场战斗不只是通过敌人，更像是一段被卡住的“连接过程”。

## 🧠 叙事方式与结构 

- 采用 **线性章节制结构**

章节结构示意：

- 第二章：一只不属于我的手
